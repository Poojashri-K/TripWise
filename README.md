# TripWise

## Description

TripWise is a smart travel planner that helps users create personalized travel itineraries based on their destination, travel dates, budget, and number of travelers. It generates day-wise travel plans, estimates trip expenses, provides weather-based recommendations, and creates a packing checklist to make trip planning simple and efficient.

## Features

- Personalized trip planning
- Day-wise travel itinerary
- Budget estimation and cost breakdown
- Weather-based travel recommendations
- Expense tracker
- Packing checklist
- Save and manage trips
- Responsive user interface
- Local storage for trip persistence

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Navigate to the project folder.
4. Install dependencies: npm install
5. Start the development server: npm run dev
6. Open the local URL displayed in the terminal.
